Apa itu Laravel?

Laravel adalah salah satu framework PHP modern yang berbasis open source. Framework ini dikhususkan untuk membangun aplikasi web interaktif. Laravel menyediakan antarmuka yang mudah digunakan dan membuat kita lebih mudah dalam membangun struktur aplikasi web. Laravel juga memiliki banyak fitur yang dapat membantu kita dalam membuat aplikasi web.

Apa yang Baru di Laravel 8?

Pada versi 8, Laravel memiliki beberapa fitur yang menarik seperti:

  • Blade Component Library
  • Livewire Component Library
  • Migration Squashing
  • CORS Support
  • Pagination Links
  • Route Model Binding Improvements
  • Job Middleware Improvements
  • Improved Rate Limiting
  • HTTP Client
  • Improved Queue Worker Output

Kenapa Memilih Laravel 8?

Laravel 8 memiliki banyak kelebihan dibandingkan versi sebelumnya. Salah satu kelebihan utama yang dimiliki oleh Laravel 8 adalah fitur yang lebih lengkap. Laravel 8 memiliki banyak fitur yang dapat membantu kita dalam membuat aplikasi web yang berkualitas. Selain itu, Laravel 8 juga memiliki kemudahan dalam membuat aplikasi web.

Bagaimana Cara Instalasi Laravel 8?

Untuk memulai proyek Laravel 8, Anda harus memiliki PHP dan Composer terinstall di sistem Anda. Setelah Anda memiliki kedua program tersebut, Anda dapat memulai instalasi Laravel 8 dengan mengetikkan perintah berikut ini di terminal Anda:

composer create-project --prefer-dist laravel/laravel nama_proyek_anda

Bagaimana Cara Membuat Route di Laravel 8?

Anda dapat membuat rute di Laravel 8 dengan menggunakan file route.php yang terletak di folder app / Http. Anda dapat menambahkan rute di file tersebut dengan menggunakan beberapa perintah seperti get, post, put, patch, dan delete. Misalnya, jika Anda ingin membuat rute untuk halaman depan Anda, Anda dapat menggunakan perintah berikut:

Route::get('/', function () { return view('home'); });

Bagaimana Cara Membuat Controller di Laravel 8?

Anda dapat membuat controller di Laravel 8 dengan menggunakan perintah berikut:

php artisan make:controller NamaController

Setelah Anda mengetikkan perintah di atas, Anda akan melihat bahwa file controller baru telah dibuat di folder app / Http / Controllers. Anda dapat membuat fungsi pada controller tersebut dengan menggunakan perintah berikut:

public function namaFungsi() { }

Bagaimana Cara Membuat View di Laravel 8?

Anda dapat membuat view di Laravel 8 dengan menggunakan perintah berikut:

php artisan make:view NamaView

Setelah Anda mengetikkan perintah di atas, Anda akan melihat bahwa file view baru telah dibuat di folder resources / views. Anda dapat membuat kode HTML di view tersebut.

Bagaimana Cara Menjalankan Aplikasi Laravel 8?

Untuk menjalankan aplikasi Laravel 8, Anda dapat menggunakan perintah berikut:

php artisan serve

Setelah Anda mengetikkan perintah di atas, Anda akan dapat menjalankan aplikasi Anda di http: // localhost: 8000. Anda juga dapat menggunakan IP Anda untuk menjalankan aplikasi Anda.
