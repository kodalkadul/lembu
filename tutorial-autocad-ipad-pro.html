Tutorial Autocad di iPad Pro, Simpel dan Mudah

Kenapa Harus di iPad Pro?

iPad Pro mempunyai segudang keunggulan dibanding tablet lainnya.Keunggulan ini dapat membantu Anda untuk mengaplikasikan Autocad. Alhasil, Anda dapat membuat desain dengan lebih cepat dan mudah.

Kualitas layar iPad Pro yang sangat tinggi dapat membantu Anda untuk melihat gambar dan desain dengan lebih jelas. Selain itu, dukungan Apple Pencil yang kuat juga dapat memudahkan Anda untuk melakukan desain.

Cara Instal Autocad di iPad Pro

Untuk memulai, Anda harus menginstal Autocad di iPad Pro Anda. Berikut ini adalah cara yang dapat Anda ikuti:

1. Buka App Store di iPad Pro Anda.

2. Cari Autocad dan klik tombol download.

3. Tunggu beberapa saat hingga Autocad selesai diunduh dan diinstal di iPad Pro Anda.

Cara Mengkonfigurasi Autocad di iPad Pro

Setelah menginstal Autocad di iPad Pro, Anda perlu mengkonfigurasi aplikasi tersebut untuk memastikan Anda dapat membuat desain dengan nyaman. Berikut ini adalah cara yang dapat Anda ikuti:

1. Klik tombol Settings di Autocad.

2. Pilih opsi Preferences.

3. Pilih opsi yang Anda inginkan untuk mengkonfigurasi Autocad.

4. Klik tombol OK untuk menyimpan perubahan.

Cara Menggunakan Autocad di iPad Pro

Setelah Anda mengkonfigurasi Autocad dengan benar, Anda dapat mulai membuat desain. Berikut ini adalah cara yang dapat Anda ikuti untuk membuat desain:

1. Klik tombol New untuk membuat proyek baru.

2. Pilih jenis desain yang ingin Anda buat.

3. Gambar desain dengan iPad Pro Anda.

4. Gunakan Apple Pencil untuk membuat desain dengan lebih detail.

5. Klik tombol Save untuk menyimpan desain.

Cara Mengelola Proyek Autocad di iPad Pro

Ketika Anda memiliki banyak proyek yang sedang dikerjakan, Anda dapat mengelola proyek dengan mudah di iPad Pro. Berikut ini adalah cara yang dapat Anda ikuti untuk mengelola proyek:

1. Klik tombol Projects di Autocad.

2. Klik pada proyek yang ingin Anda kelola.

3. Klik tombol Edit untuk mengubah proyek.

4. Klik tombol Delete untuk menghapus proyek.

5. Klik tombol Save untuk menyimpan perubahan.

Cara Mengakses Dan Berbagi Proyek Autocad di iPad Pro

Selain mengelola proyek, Anda juga dapat mengakses dan berbagi proyek di iPad Pro. Berikut ini adalah cara yang dapat Anda ikuti untuk mengakses dan berbagi proyek:

1. Klik tombol Projects di Autocad.

2. Klik tombol Share untuk berbagi proyek.

3. Masukkan alamat email orang yang ingin Anda berbagi proyek.

4. Klik tombol Send untuk mengirim proyek.

5. Klik tombol Access untuk mengakses proyek yang dibagikan.

Cara Mengelola File Autocad di iPad Pro

Ketika Anda selesai membuat desain, Anda dapat mengelola file Autocad di iPad Pro. Berikut ini adalah cara yang dapat Anda ikuti untuk mengelola file:

1. Klik tombol File di Autocad.

2. Klik tombol Rename untuk mengubah nama file.

3. Klik tombol Move untuk memindahkan file.

4. Klik tombol Copy untuk menyalin file.

5. Klik tombol Delete untuk menghapus file.

Kesimpulan

Mudah bukan? Dengan tutorial Autocad di iPad Pro ini, Anda dapat membuat desain dengan lebih cepat dan mudah. Jadi, jangan ragu untuk mencoba Autocad di iPad Pro Anda.
