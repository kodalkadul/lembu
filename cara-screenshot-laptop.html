Cara Screenshot Laptop dengan Mudah dan Cepat di Tahun 2023

Screenshot merupakan suatu cara untuk mengambil gambar dari layar laptop atau komputer. Mungkin selama ini kamu sudah sering mengambil screenshot, namun kali ini kita akan membahas cara screenshot laptop dengan mudah dan cepat di tahun 2023. Berikut adalah cara-caranya.

Cara Screenshot Seluruh Layar

Pertama, cara screenshot seluruh layar, yaitu dengan menekan tombol “Print Screen” atau “PrtScn” yang terdapat di keyboard laptop atau komputer. Setelah itu, buka aplikasi Paint atau Microsoft Word dan tekan tombol “Ctrl+V” untuk mem-paste screenshot yang sudah ditangkap. Jika sudah, simpan gambar tersebut dengan format .png, .jpg, atau .gif.

Cara Screenshot Area Terpilih

Kedua, cara screenshot area terpilih, yaitu dengan menekan tombol “Alt+Print Screen” di keyboard. Setelah itu, buka aplikasi Paint atau Microsoft Word dan tekan tombol “Ctrl+V” untuk mem-paste screenshot yang sudah ditangkap. Jika sudah, simpan gambar tersebut dengan format .png, .jpg, atau .gif.

Cara Screenshot Menggunakan Snipping Tool

Ketiga, cara screenshot menggunakan Snipping Tool. Snipping Tool adalah aplikasi pengambil screenshot yang tersedia di Windows. Caranya sangat mudah, yaitu dengan membuka aplikasi tersebut dan pilih menu “New”. Kemudian, tinggal lakukan pemilihan area yang ingin di-screenshot. Setelah itu, kamu bisa menyimpan gambar tersebut dengan format .png, .jpg, atau .gif.

Cara Screenshot Menggunakan Aplikasi Tambahan

Keempat, cara screenshot menggunakan aplikasi tambahan. Kamu dapat menggunakan aplikasi tambahan seperti Lightshot atau Snagit untuk mengambil screenshot. Caranya, unduh dan install aplikasi tersebut di laptop atau komputer kamu. Setelah itu, buka aplikasi tersebut dan lakukan pemilihan area yang ingin di-screenshot. Jika sudah, simpan gambar tersebut dengan format .png, .jpg, atau .gif.

Cara Screenshot Menggunakan Shortcut Keyboard

Kelima, cara screenshot menggunakan shortcut keyboard. Kamu dapat menggunakan shortcut keyboard untuk mengambil screenshot. Caranya, tekan tombol “Windows+Print Screen” atau “Fn+Windows+Print Screen” untuk melakukan screenshot. Setelah itu, kamu akan melihat folder “Screenshots” di folder “Pictures”. Jika sudah, simpan gambar tersebut dengan format .png, .jpg, atau .gif.

Cara Screenshot Menggunakan Aplikasi Browser

Keenam, cara screenshot menggunakan aplikasi browser. Kamu dapat menggunakan aplikasi browser seperti Chrome atau Firefox untuk mengambil screenshot. Caranya, unduh dan install aplikasi tersebut di laptop atau komputer kamu. Setelah itu, buka aplikasi tersebut dan lakukan pemilihan area yang ingin di-screenshot. Jika sudah, simpan gambar tersebut dengan format .png, .jpg, atau .gif.

Cara Screenshot dengan Menggunakan Shortcut Keyboard di Mac

Ketujuh, cara screenshot dengan menggunakan shortcut keyboard di Mac. Caranya, tekan tombol “Shift+Command+3” untuk melakukan screenshot seluruh layar, atau “Shift+Command+4” untuk melakukan screenshot area terpilih. Setelah itu, kamu akan melihat gambar yang sudah diambil di folder “Desktop”. Jika sudah, simpan gambar tersebut dengan format .png, .jpg, atau .gif.

Cara Screenshot dengan Menggunakan Aplikasi Tambahan di Mac

Kedelapan, cara screenshot dengan menggunakan aplikasi tambahan di Mac. Kamu dapat menggunakan aplikasi tambahan seperti Snagit atau Skitch untuk mengambil screenshot. Caranya, unduh dan install aplikasi tersebut di laptop atau komputer kamu. Setelah itu, buka aplikasi tersebut dan lakukan pemilihan area yang ingin di-screenshot. Jika sudah, simpan gambar tersebut dengan format .png, .jpg, atau .gif.

Cara Screenshot Menggunakan Perintah CMD

Kesembilan, cara screenshot menggunakan perintah CMD. Kamu dapat menggunakan perintah CMD untuk mengambil screenshot. Caranya, buka aplikasi CMD dan ketikkan perintah “SnippingTool.exe /clip”. Setelah itu, kamu akan melihat gambar yang sudah diambil di folder “Screenshots”. Jika sudah, simpan gambar tersebut dengan format .png, .jpg, atau .gif.

Cara Screenshot Menggunakan Aplikasi Online

Kesepuluh, cara screenshot menggunakan aplikasi online. Kamu dapat menggunakan aplikasi online seperti Awesome Screenshot atau FireShot untuk mengambil screenshot. Caranya, unduh dan install aplikasi tersebut di laptop atau komputer kamu. Setelah itu, buka aplikasi tersebut dan lakukan pemilihan area yang ingin di-screenshot. Jika sudah, simpan gambar tersebut dengan format .png, .jpg, atau .gif.

Itulah beberapa cara screenshot laptop dengan mudah dan cepat di tahun 2023. Semoga informasi ini bermanfaat dan dapat membantu kamu dalam mengambil screenshot dengan mudah dan cepat.
