Cara Mudah Cetak Quizizz dalam Bentuk PDF

Apa Itu Quizizz?

Quizizz adalah sebuah platform yang menyediakan berbagai macam kuis interaktif bagi semua orang. Aplikasi ini dapat digunakan untuk tujuan pendidikan, pembelajaran, hiburan, dan banyak lagi. Aplikasi ini dilengkapi dengan berbagai macam fitur, termasuk membuat kuis, membagikan kuis, dan membuat laporan hasil kuis. Di Quizizz, Anda juga dapat membuat kuis dalam bentuk PDF.

Mengapa Harus Cetak Quizizz dalam Bentuk PDF?

Cetak Quizizz dalam bentuk PDF memiliki beberapa keuntungan. Pertama, Anda bisa membagikan kuis dengan mudah ke orang lain. Anda tidak perlu melakukan proses upload dan download lagi. Anda juga dapat membagikan kuis dengan mudah tanpa harus mengatur ulang layout dan isinya. Kedua, Anda dapat mencetak kuis dalam bentuk PDF lebih cepat daripada jika Anda mencetaknya dalam bentuk lain. Anda juga dapat mencetak kuis dengan berbagai bentuk dan warna.

Bagaimana Cara Cetak Quizizz dalam Bentuk PDF?

Berikut ini adalah cara cetak Quizizz dalam bentuk PDF:

1. Buka Quizizz di browser Anda.

Pertama, Anda perlu membuka Quizizz di browser Anda. Anda dapat menggunakan browser apa pun yang Anda sukai, seperti Chrome, Firefox, atau Safari.

2. Masuk ke halaman "Create a Quiz".

Kemudian, masuk ke halaman "Create a Quiz" di Quizizz. Di sini, Anda dapat membuat kuis baru atau mengakses kuis yang sudah dibuat sebelumnya.

3. Pilih opsi "Print".

Di bagian atas halaman, Anda akan melihat opsi "Print". Klik opsi ini untuk membuka jendela cetak. Anda juga dapat mencetak kuis dari halaman "Dashboard" Quizizz.

4. Pilih opsi "Print to PDF".

Di jendela cetak, Anda akan melihat opsi "Print to PDF". Pilih opsi ini untuk mencetak kuis dalam bentuk PDF.

5. Klik tombol "Cetak".

Kemudian, klik tombol "Cetak" untuk mulai mencetak kuis Anda dalam bentuk PDF.

Menyimpannya di Komputer

Setelah Anda selesai mencetak kuis Anda, Anda akan diminta untuk menyimpan file PDF. Anda dapat menyimpan file tersebut di folder manapun di komputer Anda.

Menyimpan ke Google Drive

Anda juga dapat menyimpan file PDF hasil cetak Quizizz ke Google Drive. Pertama, Anda perlu masuk ke Google Drive. Kemudian, klik "Upload" dan pilih file PDF yang ingin Anda unggah. Setelah itu, file PDF akan tersimpan di Google Drive Anda.

Menyimpan ke Kelas Google

Anda juga dapat menyimpan file PDF hasil cetak Quizizz ke Kelas Google. Pertama, Anda perlu masuk ke Kelas Google. Kemudian, klik "Upload" dan pilih file PDF yang ingin Anda unggah. Setelah itu, file PDF akan tersimpan di Kelas Google Anda.

Menyimpan ke Dropbox

Anda juga dapat menyimpan file PDF hasil cetak Quizizz ke Dropbox. Pertama, Anda perlu masuk ke Dropbox. Kemudian, klik "Upload" dan pilih file PDF yang ingin Anda unggah. Setelah itu, file PDF akan tersimpan di Dropbox Anda.

Kesimpulan

Cetak Quizizz dalam bentuk PDF sangat mudah. Anda hanya perlu membuka Quizizz di browser Anda, masuk ke halaman "Create a Quiz", pilih opsi "Print", pilih opsi "Print to PDF", dan klik tombol "Cetak". Setelah itu, Anda dapat menyimpan file PDF hasil cetak di komputer, Google Drive, Kelas Google, atau Dropbox.
